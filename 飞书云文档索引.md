## 🚀 核心功能

### 1. lark-content-factory（内容工厂）

将单一输入转换为多种输出格式：
- 会议妙记 → 会议纪要 + 待办 + 新闻稿 + 社交媒体文案

### 2. lark-meeting-automation（会议自动化）

端到端会议全流程自动化：
- 会前：查询空闲 + 创建日程 + 发送邀请
- 会后：获取妙记 + 生成纪要 + 创建待办

### 3. lark-doc-enhancer（文档增强助手）

智能分析文档并添加视觉增强：
- Callout 高亮块、表格、分栏、Mermaid 画板

### 4. lark-project-setup（项目一键 Setup）

为新项目创建完整协作基础设施：
- 知识库 + 多维表格 + 项目群 + 任务模板

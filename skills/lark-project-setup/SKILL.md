# 项目一键 Setup (Project Setup)

> **前置条件：** 先阅读 [`../lark-shared/SKILL.md`](../lark-shared/SKILL.md) 了解认证和错误处理。

## 核心能力

本 Skill 为新项目提供一站式初始化服务，自动创建完整的项目协作基础设施：

```
输入：项目名称 + 基本信息
  ↓
初始化引擎
  ├── 知识库结构创建
  ├── 多维表格看板搭建
  ├── 项目群聊建立
  ├── 任务模板配置
  └── 项目手册生成
  ↓
输出：完整的项目协作环境
```

| 操作 | 所需 scope |

| 创建多维表格 | `bitable:write` |
| 创建群聊 | `im:chat` |
| 创建任务 | `task:write` |

## 项目群聊

### 群聊配置

**创建命令**:
```bash
lark-cli im +create-chat --name "{项目名称} 项目群" --members "{user_ids}"
```

**群配置**:
- 群名称：`{项目名称} 项目群`
- 群头像：项目 logo 或默认图标
- 群公告：项目目标 + 重要链接
- 群菜单：快捷入口

**群公告模板**:
```
📌 {项目名称}

🎯 项目目标：{核心目标一句话}

📅 计划周期：{开始日期} - {结束日期}

👥 项目成员：
- PM: {姓名}
- Tech Lead: {姓名}
- 开发：{名单}

🔗 重要链接：
- 知识库：{URL}
- 任务看板：{URL}
- 项目日历：{URL}

💬 群规：
1. 工作日 9:00-18:00 为响应时间
2. 紧急事项请@相关人员
3. 会议纪要自动同步到知识库
```

**群菜单配置**:
```
菜单 1: 项目文档 → 知识库 URL
菜单 2: 任务看板 → 多维表格 URL
菜单 3: 项目日历 → 日历订阅链接
```

## 使用示例

### 示例 1：新项目初始化

```
用户：帮我创建一个新项目"电商平台重构"，下周一启动，预计 3 个月完成

AI 执行流程:
1. 确认项目基本信息:
   - 项目名称：电商平台重构
   - 启动日期：下周一
   - 预计周期：3 个月
   
2. 创建知识库结构:
   - 创建知识空间
   - 创建 6 个文档分类
   - 初始化模板文档（项目章程、会议纪要等）
   
3. 创建多维表格:
   - 项目任务看板（含视图配置）
   - 需求池
   - 风险登记册
   - 项目日历
   
4. 创建项目群:
   - 创建群聊
   - 配置群公告和菜单
   
5. 创建任务模板:
   - 导入标准项目任务清单
   
6. 生成项目手册:
   - 汇总所有资源链接
   - 创建项目概览文档
   
7. 返回初始化报告
```

## 错误处理

| 错误场景 | 处理方案 |
|----------|----------|
| 用户不存在 | 跳过该用户，记录日志 |
| 权限不足 | 提示所需权限，提供授权链接 |
| 命名冲突 | 添加项目编号后缀 |
| 创建失败 | 回滚已创建内容，提供错误详情 |

## 最佳实践

### 命名规范
- ✅ 项目名 + 资源类型（如"电商平台重构 - 任务看板"）
- ✅ 文档编号前缀（如"01-项目概览"）
- ✅ 版本标记（如"v1.0"）

### 权限管理
- ✅ 最小权限原则
- ✅ 敏感文档单独设置权限
- ✅ 定期审查权限

### 维护建议
- ✅ 每周更新任务状态
- ✅ 会后 24 小时内更新纪要
- ✅ 里程碑完成后更新项目手册
